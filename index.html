One Framework, Multiple Databases

Universal Backend Engine supports multiple databases with consistent interfaces, making it easy to use the right database for each task.

// Connect to PostgreSQL
const db = await connectDatabase('postgres');

// Query data
const users = await db.query('SELECT * FROM users WHERE active = $1', [true]);

// Transaction support
await db.transaction(async (client) => {
  await client.query('UPDATE accounts SET balance = balance - $1 WHERE id = $2', [100, 1]);
  await client.query('UPDATE accounts SET balance = balance + $1 WHERE id = $2', [100, 2]);
});
// Connect to MongoDB
const db = await connectDatabase('mongodb');

// Query data
const users = await db.collection('users').find({ active: true }).toArray();

// Document operations
await db.collection('posts').insertOne({
  title: 'New Post',
  content: 'This is a new post',
  author: 'user123',
  createdAt: new Date()
});
// Connect to Redis
const redis = await connectDatabase('redis');

// Cache operations
await redis.set('user:profile:123', JSON.stringify(userProfile), 'EX', 3600);
const cachedProfile = await redis.get('user:profile:123');

// Pub/Sub
await redis.subscribe('notifications');
redis.on('message', (channel, message) => {
  console.log(`Received ${message} from ${channel}`);
});

Build Any API You Need

Universal Backend Engine supports multiple API paradigms out of the box, so you can choose the right approach for each use case.

REST API

Build traditional RESTful APIs with automatic validation, documentation, and versioning.

// Define a REST endpoint
app.get('/api/users', async (req, res) => {
  const users = await db.query('SELECT * FROM users');
  res.json(users);
});

GraphQL

Create flexible, efficient APIs with GraphQL for optimal data fetching.

// Define GraphQL schema
const typeDefs = `
  type User {
    id: ID!
    name: String!
    email: String!
    posts: [Post!]!
  }
`;

// Resolvers
const resolvers = {
  User: {
    posts: (user) => db.query('SELECT * FROM posts WHERE author_id = $1', [user.id])
  }
};

WebSockets

Build real-time applications with bidirectional communication.

// WebSocket handler
io.on('connection', (socket) => {
  socket.on('join-room', (roomId) => {
    socket.join(roomId);
    socket.to(roomId).emit('user-connected', socket.id);
  });
  
  socket.on('message', (data) => {
    io.to(data.room).emit('message', {
      sender: socket.id,
      text: data.text
    });
  });
});

gRPC

High-performance, strongly-typed APIs for microservices communication.

// gRPC service implementation
const userService = {
  getUser: async (call, callback) => {
    const { id } = call.request;
    const user = await db.query('SELECT * FROM users WHERE id = $1', [id]);
    callback(null, user);
  },
  
  createUser: async (call, callback) => {
    const { name, email } = call.request;
    const result = await db.query(
      'INSERT INTO users(name, email) VALUES($1, $2) RETURNING *',
      [name, email]
    );
    callback(null, result.rows[0]);
  }
};
